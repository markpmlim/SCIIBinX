A Mac OS X program to extract a ProDOS 8 file from a BINSCII file. It can also extract such files from BINSCII segments from a e-mail text file. The e-mail text file must be renamed with a file extension BSC or BSQ in order for the program to accept it as a valid file.


When the source codes of this project are compiled, the following programs or modules will be produced:

a) SCIIBinX.app,

b) QuickLookBSQ.qlgenerator, and,

c) SpotlightBSQ.mdimporter.


The program SCIIBinX.app demonstrates how to get a Mac OS X application to accept drags-and-drops of files.


The QuickLook module demonstrates how to return information to Mac OS X and display them as well as the generation icons for files recognized by the application.


The Spotlight module demonstrates how to return information to Mac OS X Spotlight engine.


Build requirements: XCode 3.x or later.

Runtime system requirements: 10.5 or later
